# Intarvas Communications

Intarvas Communications is a modern communications platform that provides seamless connectivity solutions for businesses and individuals.

## Project Overview

Intarvas Communications offers cutting-edge communication tools designed to enhance productivity and collaboration. Our platform integrates various communication channels into a unified interface, making it easier to stay connected.

## Features

- **IntarvAS PBX**: Bring all your communication channels together
- **All In One Solution**: End-to-end encrypted conversations
- **Bulk Messaging**: High-quality video calls with screen sharing
- **0700 & 0800**: Memorable Vanity Numbers (0700)

## Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation

```bash
# Clone the repository
git clone <YOUR_REPOSITORY_URL>

# Navigate to project directory
cd intarvas-communications

# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for Production

```bash
# Build the project
npm run build

# Preview production build
npm run preview
```

## Development

This project is built with:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - Component library
- **Tailwind CSS** - Utility-first CSS framework

## Project Structure

```
src/
├── components/     # Reusable UI components
├── lib/           # Utilities and configurations
├── pages/         # Application pages
├── styles/        # Global styles
└── App.tsx        # Main application component
```

## Deployment

This project can be deployed to various platforms:

### Vercel
```bash
npm install -g vercel
vercel
```

### Netlify
```bash
npm run build
# Drag and drop the dist folder to Netlify
```

## Environment Variables

Create a `.env` file in the root directory:

```
VITE_API_URL=your_api_url
VITE_APP_NAME=Intarvas Communications
```
